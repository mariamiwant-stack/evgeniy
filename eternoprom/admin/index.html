Мини-просмотрщик каталога

Во вкладке можно открыть любой уровень каталога, посмотреть сетку карточек, перейти в «режим бога» на живой странице, а также создать, изменить или удалить карточки без программирования. Данные автоматически синхронизируются в JSON-хранилище.

API бэкенд

Endpoint: /api/order — приём заказов из корзины
Endpoint: /api/lead — заявки на цену и формы товара
Endpoint: /api/callback — заявки на звонок (header/footer модалка)
Endpoint: /api/entries — список всех заявок (GET, только с токеном)
Бэкенд: FastAPI + SQLite. Файл: api_server.py. Запуск: python api_server.py
Настройки SMTP хранятся в api_config.json рядом с сервером.
